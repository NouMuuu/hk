# Идея
Сделать сайт про игру Hollow Knight: Silksong. Тактики против боссов.

# Референс
https://www.igromania.ru/article/33104/kak-pobedit-vdovu-v-hollow-knight-silksong-taktika-i-sovetyi/

# Основная страница
1) Кратко расписать информацию про игру
2) Сделать список всех боссов, чтобы пользователь мог нажать на нужного и его перекинуло на инструкцию к нему 
3) Добавить индикатор сложности к каждому боссу

# Функционал
1) На сайте можно будет войти в аккаунт
2) Кнопка для проматывания в самый верх страницы

# Страницы с боссами
1) Краткий рассказ как найти босса и как получить предмет для призыва(если требуется).
2) Все атаки и способы избежать их
3) Посмотреть видео с прохождением каждого босса
